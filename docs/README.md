# Документация проекта "Университетская Доска Задач"

Добро пожаловать в официальную документацию проекта **Университетская Доска Задач** — платформы для эффективного взаимодействия студентов и сотрудников университета.

## О проекте
Платформа призвана решить проблему коммуникационного разрыва между сотрудниками, которым нужна помощь в реализации проектов, и студентами, ищущими практический опыт. Система построена на принципах геймификации, прозрачности и формирования цифрового портфолио.

## Основные разделы

1.  **[Бизнес-концепция и Кейс](business_case.md)**
    Подробный разбор проблематики, целей проекта и того, как платформа решает задачи современного университета.
2.  **[Функциональные возможности](features.md)**
    Полный список реализованных фич: от модерации задач до системы достижений и защиты данных.
3.  **[Техническая архитектура](architecture.md)**
    Обзор стека технологий, структуры базы данных и организации репозитория.
4.  **[Роли и Сценарии использования](user_roles.md)**
    Описание путей пользователя для Студента, Сотрудника и Администратора.
5.  **[Обзор API](api.md)**
    Краткое руководство по эндпоинтам бэкенда.

## Быстрый запуск

### Бэкенд (FastAPI)
```bash
cd src/app
python main.py
```
Swagger UI доступен по адресу: `http://localhost:8000/docs`

### Фронтенд (React)
```bash
cd spa
npm run dev
```
Приложение доступно по адресу: `http://localhost:3000`
